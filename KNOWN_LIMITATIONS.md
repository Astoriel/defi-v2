# Known Limitations

**Snapshot date:** 2025-11-09

- Mock mode is central to reproducibility and should be disclosed as such.
- Live extractors depend on external API keys, quotas, and schema stability.
- The pipeline demonstrates modeling patterns; it is not a universal DeFi attribution product.
- Generated dbt and Evidence artifacts should be built by CI, not treated as source code.
- The churn and ROAS logic is intentionally opinionated for the demo scenario.
